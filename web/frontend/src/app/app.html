What is this?

torch.compile analyzes the neural network and fuses multiple GPU operations into optimized CUDA kernels tailored to your {{ serverStatus.gpu_name }}. This makes inference significantly faster (up to 2x) but requires a one-time compilation pass after the server starts. Once compiled, all video uploads benefit from the optimized kernels.

The pipeline also uses FP16 mixed precision to leverage your GPU's tensor cores, batch processing (16 frames at a time), and a pipelined architecture that overlaps disk I/O with GPU compute so the GPU is never idle.
